未来学園HOPE WEBサイト

WordPressオリジナルテーマによるコーポレートサイト

サイトを見る

Overview

教育機関「未来学園HOPE」のコーポレートサイトです。 WordPressのオリジナルテーマをゼロから設計・開発し、 施設の魅力を伝えつつ、運営者が簡単に更新できるサイトを構築しました。

Website Screenshot

開発の背景・動機

クライアントから「自分たちで簡単に更新できるサイトが欲しい」という 要望をいただきました。既存のテーマでは細かいカスタマイズに限界があるため、 オリジナルテーマを開発することで、デザインの自由度と運用のしやすさを 両立させることにしました。

技術的な挑戦と解決

課題：運用者が使いやすい管理画面

WordPress初心者でも迷わず更新できる管理画面が求められました。

解決策

カスタムフィールド（ACF）を活用し、必要な項目だけを わかりやすく入力できるフォームを作成。 マニュアルなしでも直感的に操作できる管理画面を実現しました。

課題：デザインの再現性

デザインカンプ通りの見た目を、様々なデバイスで崩れなく表示する 必要がありました。

解決策

CSS設計にBEM命名規則を採用し、コンポーネント単位で管理。 Flexbox/Gridを活用したモダンなレイアウトで、 保守性の高いコードを実現しました。

使用技術

CMS WordPress
言語 PHP / HTML / CSS / JavaScript
プラグイン ACF, Yoast SEO
開発期間 約2ヶ月

学んだこと

このプロジェクトを通じて、「作って終わり」ではなく「運用を見据えた設計」の 重要性を学びました。クライアントとのコミュニケーションを重ね、 本当に必要な機能を見極めることの大切さを実感しました。

また、WordPressのテーマ開発を通じて、PHPの理解が深まり、 サーバーサイドの仕組みについても学ぶことができました。
