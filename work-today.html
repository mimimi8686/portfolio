Overview

「今日なにする？」は、休日や空き時間に「何をしようか迷う」という 日常的な悩みを解決するアプリです。ユーザーの気分や状況に合わせて、 やることを提案します。

App Screenshot（開発中）

※ 現在開発中 - WEB版・iOS版を同時開発

開発の背景・動機

せっかくの休日なのに「何しよう…」とスマホをダラダラ見て終わってしまう。 そんな経験は誰にでもあるはず。

「決断疲れ」を解消し、気軽に新しいことにチャレンジするきっかけを 提供できるアプリを作りたいと考えました。

予定している機能

  • 🎲

    ランダム提案

    ボタン一つで今日やることをランダムに提案

  • 😊

    気分に合わせた提案

    「アクティブ」「まったり」など気分を選んで提案を絞り込み

  • 📝

    カスタムリスト

    自分だけの「やりたいことリスト」を登録

  • 📱

    マルチプラットフォーム

    WEB版とiOS版を提供、どこからでもアクセス可能

開発アプローチ

AIアシスタント（ChatGPT）を活用した開発

このプロジェクトでは、ChatGPTに質問しながら実装を進めています。 わからない部分を質問し、返ってきたコードを理解・検証しながら 開発を進めるスタイルです。

私が担当していること

・アプリの企画・設計（何を作るか、どんなUIにするか）
・AIの回答を理解し、正しく動くか検証
・エラーが出た時のデバッグと問題解決
・UIの調整やユーザー体験の改善

マルチプラットフォーム開発への挑戦

同じアプリをWEB版とiOS版で提供することで、 異なる技術スタック（Web技術とSwift）の両方を学んでいます。

使用技術

iOS版 Swift / SwiftUI
WEB版 HTML / CSS / JavaScript
開発支援 ChatGPT
デザイン Figma

学んでいること

このプロジェクトを通じて、「同じ機能を異なるプラットフォームで実装する」 経験を積んでいます。それぞれの言語・フレームワークの違いを理解し、 適切な実装方法を選択する力を養っています。

また、AIに質問する際の「良い質問の仕方」を学ぶことで、 より効率的に問題解決できるようになりました。
